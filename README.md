# GrowAI 🚀

An AI-powered SaaS platform for social media creators — caption generation, content ideas, link-in-bio monetization, and analytics.

## Features

- 🤖 AI caption generator (OpenAI GPT-4o)
- 💡 Content ideas engine
- 🔗 Link-in-bio monetization pages
- 📊 Creator analytics dashboard
- 💳 Freemium model with Stripe billing
- 🔐 Auth via Supabase

## Tech Stack (Planned)

| Layer | Tech |
|---|---|
| Frontend | Next.js 14, Tailwind CSS |
| Backend / DB | Supabase (Postgres + Auth) |
| AI | OpenAI GPT-4o API |
| Payments | Stripe |
| Hosting | Vercel |

## Current State

This repo contains the marketing landing page and auth UI (HTML/CSS/JS). Full Next.js app in development.

## Pages

- `index.html` — Landing page
- `pages/pricing.html` — Pricing page (Free / Pro / Agency)
- `pages/login.html` — Sign in / Sign up
